# Game Chat

When players join the game they should be automatically be in the chat.

## Basics

Only special functions that should be there:

* the dice rolls should be seen there for all players
* there is an option to write something only seen by the Admin
* each message is written with the name of the player and change of name is possible

### Names

* the name of the player is used for that game only (not the login user name)
* it should be very easy to change the used name (bottom of the chat?)
* User name is used only as a fallback option (didn't change or deleted the name)
