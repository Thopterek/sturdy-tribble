# Map Building

Using reusable components (pixel art/simple drawing).

Main idea is to give option for GM (Game Master -> Game admin).

Building a top down view for other players to help with understanding of surroundings.

## Types of Components

Very basic rectangles that would represent:

* Door
* Chest
* Road
* Table
* Tree
* Water
* Rock
* Wall
* Bed

### Usage of the Components

It should be as easy as drag and drop or self-explanatory.

Some examples as to how it could be implemented:

* Pop up menu.
* Extension of the sidebar / top bar.
* [Example of the steam game that does it over the top is here](https://store.steampowered.com/app/2498570/Canvas_of_Kings/)
